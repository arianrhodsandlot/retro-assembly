<h1 align="center">Retro Assembly</h1>

## Overview
A personal retro game collection cabinet in your browser.

## Usage
1. Visit [retro-assembly.pages.dev](https://retro-assembly.pages.dev)
2. Pick a method you prefer to provide your ROMs
3. Start to play!
